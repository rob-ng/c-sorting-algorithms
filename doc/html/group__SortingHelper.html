Find median of three elements in given array and return its index.

When attempting to find a pivot point (e.g. in quicksort), the median is a better option than using a fixed position (e.g. always first element).

Parameters
arrArray containing the elements.
sizeSize of each element.
aFirst element.
bSecond element.
cThird element.

Index of the median element.
